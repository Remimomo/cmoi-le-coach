# Garmin officiel

Cette architecture prépare la future connexion Garmin sans utiliser de méthode non officielle.

Garmin Connect Developer Program donne accès aux API officielles après validation. Les API utiles pour C'moiLeCoach seront :

- Health API : sommeil, fréquence cardiaque, stress, Body Battery, métriques de récupération selon accès accordé
- Activity API : activités sportives et détails d'entraînement
- Training API : future possibilité d'envoyer des séances structurées vers Garmin

Tant que l'accès partenaire n'est pas validé, l'application continue d'utiliser les données Garmin simulées.
